#  Image Watermarking Desktop App
## Concepts Practised
- Creating Windows and Labels with Tkinter
- Use grid() and columnspan to Complete the User Interface
- Dialog Boxes and Pop-Ups in Tkinter
- Selecting Files to Watermark
- Saving Data to File
